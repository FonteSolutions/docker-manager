# Angular 2 Electron Brscan Starterkit

Estrutura para apps [electron] com [angular] usando [Webpack], [ngrx] e [material2]

Para rodar em dev:

## Run the example

```bash
$ npm install
$ npm run watch
$ npm run electron
```

## Packaging

Para gerar build

```bash
$ npm run build-package
```
